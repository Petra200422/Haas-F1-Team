# Haas-F1-Team
Završni rad


## Radnje s githubom

```
git init
git config user.email
git config user.name
git clone
cd

```

## Instaliranje quasara

```
npm i -g @quasar/cli
npm init quasar
cd 
```
Pri inicijalizaciji frontend-a i backend-a potrebno je preuzeti sve potrebne module

## Frontend

```
npm install
quasar dev
```

## Backend

```
npm install
node api.js / nodemon api.js
```
