## Radnje s githubom

```
git init
git config user.email
git config user.name
git clone
cd pa dalje
```

Pri inicijalizaciji frontend-a i backend-a potrebno je preuzeti sve potrebne module



## Frontend

```
npm install
quasar dev
```

## Backend

```
npm install
node api.js / nodemon api.js
```
